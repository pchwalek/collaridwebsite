# CollarID Website

A high-end, responsive website template for the CollarID wildlife tracking and monitoring system.

## Features

- **Dark Mode Design**: Modern dark theme with glassmorphism effects
- **Smooth Animations**: GSAP-powered scroll animations and transitions
- **Responsive Layout**: Fully responsive design using Tailwind CSS
- **Modern UI**: Lucide icons and gradient effects for a professional look

## Sections

1. **Hero**: "A New Lens on Life" with video background
2. **Mission**: Contrast between location-only data and holistic sensing
3. **Hardware Specs**: Engineering details including FEA validation
4. **Current Status**: Mk II evaluation timeline and deployments
5. **Dashboard**: Placeholder for real-time data visualization
6. **Contact**: Email link to patrick@patrickchwalek.com

## Technologies

- HTML5
- Tailwind CSS (CDN)
- Lucide Icons
- GSAP (GreenSock Animation Platform)
- ScrollTrigger

## Media Files

All media files are located in the `media/collarid/` directory and include:
- Exploded view videos
- Deployment images
- FEA simulation videos
- Acoustic recordings and spectrograms

## Usage

Simply open `index.html` in a web browser. For best results, serve via a local web server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.
